Bugs

This is a good tab to put new bugs into. If you think you've identified a reproducable problem with the Ruby API, follow these steps:

  1. Locate this tab's folder in <YOUR SKETCHUP>/Plugins/Developer/testup/tests/bugs/
  2. Make a copy of TC_NEW_BUG.rb and give it a nice name, like TC_RubyLoadBug.rb
  3. Follow the example code inside your new file to reproduce the bug.
  4. Yay! Alert the SketchUp team. We got some work to do.
